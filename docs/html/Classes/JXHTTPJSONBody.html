Inherits from NSObject
Conforms to JXHTTPRequestBody
Declared in JXHTTPJSONBody.h

Overview

JXHTTPJSONBody conforms to JXHTTPRequestBody and provides functionality for JSON POST request bodies.

Reports its content type as application/json; charset=utf-8.
